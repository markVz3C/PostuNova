Acerca del Proyecto

Screen Balance es una aplicación innovadora diseñada para ayudar a estudiantes y educadores a mantener un equilibrio saludable en el uso de dispositivos digitales, promoviendo hábitos responsables y mejorando el bienestar general.

Uso del Proyecto

Screen Balance ofrece múltiples funcionalidades para diferentes tipos de usuarios, desde estudiantes hasta educadores y padres de familia.
