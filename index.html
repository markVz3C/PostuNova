Acerca del Proyecto

Screen Balance es una aplicación innovadora diseñada para ayudar a estudiantes y educadores a mantener un equilibrio saludable en el uso de dispositivos digitales.

Uso del Proyecto

Screen Balance ofrece múltiples funcionalidades para diferentes tipos de usuarios, desde estudiantes hasta educadores y padres de familia.

Para Estudiantes

Mejoras para el Usuario

Nuestro proyecto implementa mejoras continuas basadas en investigación científica y feedback de usuarios.

Filtros de luz azul, recordatorios de parpadeo y ejercicios oculares automatizados.

Integración con dispositivos wearables y sugerencias de ejercicios durante los descansos.
